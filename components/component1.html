Navigation

This should be applied as the top navigation bar for all pages. It should remain at the top of the viewport, and the logo should be linked to the homepage.

Accessibility Considerations

This navigation bar uses dark text against a light background. If using a dark background, then light text should be used to preserve color contrast standards. The color scheme used in this component meets the WCAG color contrast requirement, and it is certified color blind safe in Adobe Color.
