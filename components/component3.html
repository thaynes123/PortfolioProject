Footer

The footer displays navigation, social media & contact links, and copyright information. The footer must remain at the bottom of every page.

Accessibility Considerations

The semantic footer element helps assistive technologies (screen readers, voice navigation) understand that this section contains site-level information and navigation. This improves page structure for anyone using screen readers or browsers with accessibility settings.
Icons alone are not accessible because screen readers can’t infer meaning visually, and using the aria-label gives a text alternative, so users hear “LinkedIn” instead of “link.”
